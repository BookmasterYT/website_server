What is the Cramped SMP?

The Cramped SMP is a server with a 1000x1000 world border. It also utilizes the Towny plugin to give the capability to join a nation and create towns. it is Java/Bedrock compatible, meaning both Java and Bedrock players can join the server! Click the "Join" section to learn how to join the server on both Java and Bedrock!
